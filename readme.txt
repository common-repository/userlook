== Installation ==

Follow simple steps to add [UserLook](http://userlook.com) tracking to your blog.

1. Upload plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Open plugin settings and fill required fields, only your blog url and site ID from UserLook required for initial setup.
